Tool Guide

Rainbow Brush

Paints with a new color everytime when the cell is white.
When the cell already has a color it darkens the existing color.

Mono Color Brush

Colors a cell with your chosen color and can repaint an already colored cell.
Pro Tip: If you want to change the mono color of some cells into rainbow color first paint them white with the mono brush and then use the rainbow brush.
